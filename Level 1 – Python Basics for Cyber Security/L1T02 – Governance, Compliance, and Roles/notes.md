# Cyber-security Governance

## Cyber Security Governance

1. Defines regulations that guide a business' overall strategic objectives and promotes awareness of- and an effective approach to dealing with security risks.

2. This is done by providing framework that outlines how organisations:

- Enforces Security Measures
- Calculates operational risk areas
- Determines responsible stakeholders

1. Ensures clear accountability, and provides appropriate steps to accountable parties to deal manage risk effectively.

## Principles-based Governance

> [!NOTE]
> Principles-based governance strategies reduce the dependence on cyber security standards by organisations like ISO[^1].

### Benefits

- Efficient cyber security issue incidence response.
- Rapid identification of new risks.
- A collection of high-quality test data for future security refinements.
- Fostering a company-wide awareness of cyber security threats.

[^1]: International Organisation for Standardization

### Commonly Used Compliance Frameworks

- **NIST Cybersecurity Framework**
  - Encourages voluntary collaboration between the public and private sectors.
  - Helps organisations identify and manage cyber security threats.
  - Considered one of the most reputable and widely adopted frameworks.

- **General Data Protection Regulation (GDPR)**
  - Introduced in 2016 to strengthen data protection for EU citizens.
  - Applies to any organisation (including US businesses) that collects personal data from EU citizens.
  - Defines requirements for consumer data rights, data protection, and breach‑notification protocols.

- **IoTSF Security Compliance Framework**
  - Developed by the Internet of Things Security Foundation, a global non‑profit.
  - Brings together IoT security experts, network providers, vendors, distributors, insurers, and government bodies.
  - Specifies security requirements for business systems, mobile apps, and cloud services.

> [!NOTE]  
> Many other compliance frameworks exist, all designed to regulate unethical online behaviour. As you grow as a cyber security professional, your decisions should align with relevant frameworks, as violations can lead to serious legal or financial consequences.

### System Administration

System administrator responsibilities within an organisation:

- Managing software
- Licensing software
- Maintaining/Updating software
- Database administration (potentially)
- Managing Hardware
- Managing Web Servers (if applicable)

> [!IMPORTANT]
  > Sysadmins are responsible for employing strategies to address and resolve issues like `server downtime` and `zero-day attacks`.

> [!NOTE]
> A system administrator is also called **sysadmin**/**sys admin**.
